# AI 修复指令：分镜生图/生视频预览与持久化问题修复

## 问题分析

### 问题1：分镜生图预览不显示
**原因**: 当前代码缺少对异步任务的实时状态展示，用户无法看到生成进度。

### 问题2：分镜生视频无预览
**原因**: `handleGenerateVideo()` 函数仅提交任务但未处理结果，缺少视频预览框和轮询逻辑。

### 问题3：数据持久化不足
**原因**: 生成的图片/视频仅存在内存状态中，切换页面或刷新会丢失。

---

## 修复方案

const [currentTaskId, setCurrentTaskId] = useState<string | null>(null);

// 修改 handleGenerateImage 函数（第57-150行）
const handleGenerateImage = async () => {
  if (!editablePrompt) return;
  setGeneratingImg(true);
  setProgress(10);
  setProgressText('正在提交任务...');

      }),
    });
    const data = await res.json();
    console.log('[StoryboardCard] API返回:', data);
    
    const imageUrls = data?.data?.imageUrls || data?.imageUrls;
    const taskId = data?.data?.taskId || data?.taskId;
    
    if (imageUrls && imageUrls.length > 0) {
      const url = imageUrls[0];
      console.log('[StoryboardCard] 同步模式获取图片:', url);

      onImageGenerated?.(url);
      setProgress(100);
      setProgressText('生成完成');
      toast.success('分镜图生成成功');
      // 自动保存到数据库
      await handleAutoSave(url);
    } else if (taskId) {
      console.log('[StoryboardCard] 异步模式，任务ID:', taskId);
      setCurrentTaskId(taskId);
      setProgress(20);
      setProgressText('任务已提交，等待处理...');
      
      // 立即开始轮询（不延迟）
      pollImageTask(taskId);
    } else {
      const errorMsg = data?.error || data?.data?.error || '生图失败，未知响应格式';
      console.error('[StoryboardCard] 生图失败:', errorMsg, data);
      toast.error(errorMsg);
      setGeneratingImg(false);
      setProgress(null);
    }
  } catch (error) {
    console.error('生图错误:', error);
    toast.error('生图失败，请检查网络和API配置');
    setGeneratingImg(false);
    setProgress(null);
  }
};

// 新增轮询函数
const pollImageTask = async (taskId: string, attempts: number = 0) => {
  const maxAttempts = 60;
  const pollInterval = 3000;
  
  try {
    const taskRes = await apiFetch(`/api/ai/tasks/${taskId}`);
    const taskData = await taskRes.json();
    console.log(`[StoryboardCard] 轮询第${attempts + 1}次, 状态:`, taskData?.data?.status);
    
    const status = taskData?.data?.status;
    const progressPercent = Math.min(20 + (attempts * 1.5), 90);
    
    if (status === 'SUCCEEDED' || status === 'succeeded' || status === 'SUCCESS') {
      const urls = taskData?.data?.output?.imageUrls || 
                   taskData?.data?.imageUrls || [];
      
      if (urls.length > 0) {
        const url = urls[0];
        console.log('[StoryboardCard] 任务完成，获取图片:', url);
        setImgUrl(url);
        onImageGenerated?.(url);
        setProgress(100);
        setProgressText('生成完成');
        toast.success('分镜图生成成功');
        // 自动保存到数据库
        await handleAutoSave(url);
      } else {
        toast.error('任务完成但未获取到图片');
      }
      setGeneratingImg(false);
      setCurrentTaskId(null);
      setTimeout(() => setProgress(null), 2000);
    } else if (status === 'FAILED' || status === 'failed') {
      toast.error(taskData?.data?.message || '图片生成失败');
      setGeneratingImg(false);
      setCurrentTaskId(null);
      setProgress(null);
    } else {
      // 更新进度
      setProgress(Math.round(progressPercent));
      setProgressText(`正在生成... ${Math.round(progressPercent)}%`);
      
      if (attempts < maxAttempts) {
        setTimeout(() => pollImageTask(taskId, attempts + 1), pollInterval);
      } else {
        toast.error('图片生成超时，请稍后刷新查看');
        setGeneratingImg(false);
        setCurrentTaskId(null);
        setProgress(null);
      }
    }
  } catch (err) {
    console.error('[StoryboardCard] 轮询出错:', err);
    if (attempts < maxAttempts) {
      setTimeout(() => pollImageTask(taskId, attempts + 1), pollInterval);
    } else {
      toast.error('查询任务状态失败');
      setGeneratingImg(false);
      setCurrentTaskId(null);
      setProgress(null);
    }
  }
};

// 新增自动保存函数
const handleAutoSave = async (url: string) => {
  try {
    const res = await apiFetch('/api/assets', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        type: 'image',
        category: 'storyboard',
        name: `分镜${sb.num || index + 1}`,
        url: url,
        prompt: editablePrompt,
        model: 'seedream-4.6-t2i',
        metadata: {
          index: index,
          ratio: imgRatio,
        },
      }),
    });
    const result = await res.json();
    if (!result.success) {
      console.warn('自动保存失败:', result.error);
    }
  } catch (err) {
    console.warn('自动保存失败:', err);
  }
};

# AI 修复指令：分镜页面优化提示词保存与运镜硬编码问题

## 问题分析

### 问题1：优化提示词不保存
**原因**: `handleOptimizePrompt()` 函数仅更新组件内存状态，未持久化到数据库，切换页面后丢失。

### 问题2：运镜硬编码
**原因**: 分镜数据中的 `camera` 字段可能默认为"固定"字符串，属于硬编码值。

---

## 修复方案

### 步骤1：修改 StoryboardCard.tsx - 优化提示词自动保存

**文件**: [`src/components/StoryboardCard.tsx`](file:///d:/nvwa_projects/src/components/StoryboardCard.tsx)

**修改内容**: 在 `handleOptimizePrompt` 函数中添加自动保存逻辑

```typescript
// 修改 handleOptimizePrompt 函数（第39-55行）
const handleOptimizePrompt = async () => {
  setOptimizing(true);
  try {
    const res = await apiFetch('/api/ai/optimize-prompt', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ prompt: editablePrompt }),
    });
    const data = await res.json();
    if (data.success && data.data) {
      const optimizedPrompt = String(data.data);
      setEditablePrompt(optimizedPrompt);
      
      // 自动保存优化后的提示词到数据库
      await saveOptimizedPrompt(optimizedPrompt);
      
      toast.success('提示词优化成功');
    }
  } catch {
    // keep current prompt
  }
  setOptimizing(false);
};

// 新增保存提示词函数
const saveOptimizedPrompt = async (prompt: string) => {
  try {
    const res = await apiFetch('/api/assets', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        type: 'text',
        category: 'storyboard-prompt',
        name: `分镜${sb.num || index + 1}提示词`,
        url: '',
        prompt: prompt,
        metadata: {
          index: index,
          originalPrompt: sb.description,
        },
      }),
    });
    const result = await res.json();
    if (!result.success) {
      console.warn('保存优化提示词失败:', result.error);
    }
  } catch (err) {
    console.warn('保存优化提示词失败:', err);
  }
};
```

// 在加载分镜数据后，尝试加载优化提示词
useEffect(() => {
  const loadOptimizedPrompts = async () => {
    try {
      const authHeaders = getAuthHeaders();
      const response = await fetch('/api/assets?category=storyboard-prompt', {
        headers: authHeaders,
      });
      const result = await response.json();
      
      if (result.success && result.data) {
        const prompts = result.data as Array<{
          id: string;
          prompt: string;
          metadata: { index: number };
        }>;
        
        // 更新分镜的优化提示词
        setStoryboards(prev => prev.map((sb, index) => {
          const savedPrompt = prompts.find(p => p.metadata?.index === index);
          if (savedPrompt) {
            return { ...sb, description: savedPrompt.prompt };
          }
          return sb;
        }));
      }
    } catch (err) {
      console.warn('加载优化提示词失败:', err);
    }
  };
