
## 问题分析

### 1. 分镜生成功能的数据流

**后端API (`/api/ai/split-scenes`)**：
- 返回 SSE (Server-Sent Events) 流式响应
- 流式 chunk: `{ content: "..." }`
- 最终 complete 事件: `{ type: "complete", data: { scenes: [...] }, message: "..." }`

**useAIStream Hook**：
- 接收流式响应
- 最终返回 `finalResult = data.data` → `{ scenes: [...] }`

**前端剧本详情页面**：
- `handleSplitScenes` 函数调用 `splitStream.run()`
- 期望接收 `{ scenes: [...] }` 格式
- 然后转换为 `ScriptStoryboard[]` 格式

### 2. 问题根源

经过分析，代码结构是正确的，但可能存在以下问题：

**问题1：useAIStream 返回的数据结构可能被错误处理**
在 `useAIStream.ts` 第 147 行：
```typescript
finalResult = data.data ?? (accumulatedText ? { content: accumulatedText } : data);
```

**问题2：API 的 scenes 数据可能为空**
如果 LLM 解析失败，`scenes` 可能是空数组

**问题3：前端数据转换逻辑可能有遗漏**

**修复代码**:
```typescript
const handleSplitScenes = async () => {
  if (!script?.content?.trim()) { 
    toast.error("剧本内容为空，请先在剧本编辑区输入内容");
    setActiveView("storyboard");
    return; 
  }
  // 内容较短时直接生成
  if (script.content.length < 5000) {
    setIsSplitting(true);
    setActiveView("storyboard");
    setShowEpisodeSelector(false);
    try {
      const result = await splitStream.run("/api/ai/split-scenes", {
        scriptContent: script.content,
      });
      
      // 调试日志
      console.log("split-scenes 返回结果:", result);
      
      const data = result as Record<string, unknown>;
      let scenes: ScriptStoryboard[] = [];
      
      // 尝试从 data.scenes 获取
      if (Array.isArray(data.scenes)) {
        scenes = (data.scenes as Record<string, string>[]).map((s, i) => ({
          num: i + 1,
          duration: s.duration || s.time || "5s",
          description: s.description || s.scene || s.content || "",
          character: s.character || s.characters || "",
          shot: s.shot || s.type || "中景",
          camera: s.camera || s.movement || "固定",
          audio: s.audio || s.sound || "",
        }));
      }
      
      // 如果 scenes 为空，尝试从 data.content 获取
      if (scenes.length === 0 && data.content) {
        let raw = data.content;
        if (typeof raw === 'string') {
          try { 
            raw = JSON.parse(raw); 
          } catch (err) {
            console.error("JSON解析失败:", err);
            // 尝试提取数组
            const arrayMatch = raw.match(/\[[\s\S]*\]/);
            if (arrayMatch) {
              try {
                raw = JSON.parse(arrayMatch[0]);
              } catch {
                // 失败则使用原始字符串
              }
            }
          }
        }
        if (Array.isArray(raw)) {
          scenes = (raw as Record<string, string>[]).map((s, i) => ({
            num: i + 1,
            duration: s.duration || s.time || "5s",
            description: s.description || s.scene || s.content || "",
            character: s.character || s.characters || "",
            shot: s.shot || s.type || "中景",
            camera: s.camera || s.movement || "固定",
            audio: s.audio || s.sound || "",
          }));
        }
      }
      
      console.log("最终解析的分镜数据:", scenes);
      
      setScript((prev) => (prev ? { ...prev, storyboards: scenes } : null));
      try {
        await apiFetch(`/api/scripts/${id}`, {
          method: "PUT",
          headers: { "Content-Type": "application/json" },
          body: JSON.stringify({ storyboards: scenes }),
        });
      } catch {
        toast.error("分镜数据保存到数据库失败");
      }
      if (scenes.length > 0) {
        toast.success(`成功拆分出 ${scenes.length} 个分镜`);
      } else {
        toast.error("分镜拆分结果为空，请尝试重新生成");
      }
    } catch (err) {
      console.error("分镜拆分错误:", err);
      toast.error("分镜拆分失败");
    } finally {
      setIsSplitting(false);
    }
  } else {
    // 内容较长，先分析集数
    handleAnalyzeEpisodes();
  }
};
```
