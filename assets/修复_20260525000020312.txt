#### 1.2 数据库缺少封面图字段

**问题**：封面图无法持久化存储，切换页面或登出后消失

**根本原因**：
- API尝试保存 `coverImage` → `cover_image`
- 但数据库schema中没有 `cover_image` 字段
- 导致封面图保存被静默忽略

**修复方案1 - 添加数据库字段**：

在 Supabase 数据库中执行：

```sql
ALTER TABLE scripts ADD COLUMN IF NOT EXISTS cover_image TEXT DEFAULT '';
```

**修复方案2 - 修改schema文件**：`/src/storage/database/shared/schema.ts`

```typescript
export const scripts = pgTable("scripts", {
  // ... 现有字段 ...
  
  // 添加封面图字段（在 episodeCount 之后）
  episodeCount: integer("episode_count").default(0),
  coverImage: text("cover_image").default("").notNull(),  // 新增
  wordCount: integer("word_count").default(0),
  sceneCount: integer("scene_count").default(0),
}, (table) => [
  // ... 现有索引 ...
]);
```

**验证修复**：
1. 生成封面图
2. 刷新页面
3. 验证封面图仍然显示
4. 登出并重新登录
5. 验证封面图仍然显示

#### 3.1 生产环境日志控制

**问题**：大量 `console.log` 在生产环境可能泄露敏感信息

**建议**：在关键API中添加日志级别控制

```typescript
// 创建日志工具：/src/lib/logger.ts
const LOG_LEVEL = process.env.NODE_ENV === "production" ? "error" : "debug";

export const logger = {
  debug: (...args: unknown[]) => {
    if (LOG_LEVEL === "debug") console.log("[DEBUG]", ...args);
  },
  info: (...args: unknown[]) => {
    if (LOG_LEVEL !== "silent") console.log("[INFO]", ...args);
  },
  error: (...args: unknown[]) => {
    console.error("[ERROR]", ...args);
  },
};
```

**应用到关键文件**：

```typescript
// /src/app/api/ai/generate-image/route.ts
// 将 console.log 替换为 logger.debug

import { logger } from "@/lib/logger";

// 修改前
console.log("[Generate Image API] Using API Key:", apiKey?.substring(0, 10) + "...");

// 修改后
logger.debug("[Generate Image API] Using API Key:", apiKey?.substring(0, 10) + "...");
```
