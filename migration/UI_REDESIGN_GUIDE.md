# 女娲平台 UI 重构指南

## 一、项目概述

**女娲** - AI智能短剧创作平台，支持从创意到成片的全流程自动化创作。

### 核心定位
- **目标用户**：短剧创作者、编剧、导演、影视从业者
- **核心价值**：AI驱动的剧本生成、角色设计、分镜创作、视频生成
- **视觉风格**：高级AI智能创作平台，具有视觉冲击力

---

## 二、保留内容（不可修改）

### 1. 首页（`/`）
**完全保留**，不做任何修改：
- 全屏视频轮播背景
- 深色主题 + 蒂芙尼蓝（#0ABAB5）强调色
- "一念成剧 万象生辉" 标题
- "立即创作" CTA按钮
- 右侧视频预览指示器

### 2. 后端 API 接口（全部保留）
所有 `/api/*` 接口逻辑完全保留，只重构前端UI。

### 3. 数据库结构
所有数据库表结构保留（见 `migration/schema.sql`）。

### 4. 核心功能逻辑
- AI 生图/生视频调用逻辑
- 剧本解析和生成逻辑
- 角色一致性锚点逻辑
- 积分计费逻辑

## 四、新 UI 设计方向

### 1. 视觉风格
- **主题**：高级AI智能创作平台
- **风格参考**：Runway ML、Midjourney、Pika Labs
- **关键词**：沉浸式、视觉冲击、动态交互、专业感

### 2. 配色方案
```css
/* 延续首页风格 */
--background: #0A0A0A;           /* 极深黑 */
--surface: #141414;              /* 卡片背景 */
--surface-elevated: #1A1A1A;     /* 悬浮卡片 */
--primary: #0ABAB5;              /* 蒂芙尼蓝（强调色） */
--primary-glow: rgba(10, 186, 181, 0.3); /* 发光效果 */
--text-primary: #F5F5F5;         /* 主文字 */
--text-secondary: #888888;       /* 次文字 */
--border: #333333;               /* 边框 */
```

### 3. 核心组件
- **Hero区域**：大图/视频背景 + 标题 + CTA
- **功能卡片**：悬浮卡片 + 发光边框 + 悬停动效
- **数据展示**：动态图表 + 实时更新 + 进度动画
- **创作画布**：全屏沉浸式 + 工具栏悬浮
- **表单**：玻璃拟态 + 分组清晰

### 4. 动效设计
- 页面切换：淡入淡出 + 滑动
- 卡片悬停：发光边框 + 微上浮
- 按钮交互：涟漪效果 + 缩放
- 数据加载：骨架屏 + 渐入
- 状态变化：平滑过渡

## 五、页面重构详细规格

### 1. Dashboard（仪表盘）

**当前**：后台系统风格，数据卡片堆叠
**目标**：创作中心首页

**布局**：
```
┌─────────────────────────────────────┐
│  Hero: 欢迎回来 + 快捷创作按钮         │
├─────────────────────────────────────┤
│  统计卡片区（横向4卡片）               │
│  - 创作剧本数  - 角色数  - 资产数  - 积分 │
├─────────────────────────────────────┤
│  最近创作（卡片轮播）                  │
│  - 剧本卡片  - 角色卡片  - 视频卡片     │
├─────────────────────────────────────┤
│  快捷入口（大图标卡片）                │
│  - AI生图  - AI生视频  - 剧本创作       │
└─────────────────────────────────────┘
```

## 六、技术约束

### 必须保持
1. **API 接口路径不变**：所有 `/api/*` 路由保持不变
2. **数据结构不变**：数据库表和字段名不变
3. **功能逻辑不变**：业务逻辑代码可复用
4. **首页不变**：`/` 页面完全保留

### 可以修改
1. 页面布局和组件结构
2. CSS 样式和视觉效果
3. 组件文件拆分和命名
4. 交互方式和动效

### 开发规范
- 使用 Next.js 16 App Router
- 使用 React 19 + TypeScript 5
- 使用 Tailwind CSS 4
- 使用 shadcn/ui 组件库
- 仅使用 pnpm 管理依赖
